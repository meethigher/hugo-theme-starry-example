# Hugo Theme Starry Example

This is an example repository for the **Starry** theme, demonstrating how to use Starry in a Hugo project.

## Directory Structure

```
hugo-theme-starry-example/
├── content/        # Example Markdown posts
├── README.md
```

## Quick Start

Install Hugo (v0.154.0 or above recommended)

```sh
git clone https://github.com/meethigher/hugo-theme-starry-example.git
cd hugo-theme-starry-example
git clone https://github.com/meethigher/hugo-theme-starry.git themes/starry
cp themes/starry/hugo-example.toml hugo.toml
hugo server
```
